# College Mapping System - SSASIT (Design Thinking Project)

This application is the digital implementation of a Design Thinking project aimed at solving campus navigation challenges at the **Shree Swami Atmanand Saraswati Institute of Technology (SSASIT), Surat**.

## 📖 Overview

Navigating a complex campus environment can be overwhelming, especially for new students, parents, and visitors. Our project, "College Mapping System," originated from thorough field research and Design Thinking methodologies to create a structured, accessible, and intuitive way to understand and navigate campus facilities.

The core objective was to eliminate confusion, reduce time lost searching for classrooms or offices, and provide a centralized, digital map that bridges academic learning with industry-standard web development practices.

## 🎓 Design Thinking Foundation

This project was developed by **Team 948876** (Information Technology, B.E. 1st Year, 2nd Semester, 2026 Batch) under the guidance of our Mechanical Engineering Department faculty.

The solution was derived through several stages of the Design Thinking process:
- **Empathize & Define:** Used AEIOU canvases and Mind Mapping to identify user pain points (e.g., lack of clear signage, confusing building layouts).
- **Ideate & Prototype:** Conducted "Ideahunt" workshops and developed paper-based prototypes to validate navigation flows.
- **Product Development:** Iterated through 4 versions of a product development canvas to ensure feature relevance (interactive maps, facility markers).

## 🚀 Key Features

*   **Interactive Campus Maps:** Visual layouts for the 1st, 2nd, and 4th floors.
*   **Intuitive Navigation:** Built with `react-zoom-pan-pinch` for seamless zooming, panning, and interaction.
*   **Facility Markers:** Easy identification of classrooms, laboratories, H.O.D. offices, and amenities.
*   **Canvas Gallery:** Integrated gallery to view original project documentation, sketches, and reference signage.
*   **Responsive & Dynamic:** Developed to work smoothly on both desktop and mobile devices.

## 🛠 Tech Stack

Our implementation leverages modern web technologies for performance and maintainability:

- **Frontend:** React 18+
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Interactivity:** `react-zoom-pan-pinch` (for map manipulation), `motion` (for animations)
- **Language:** TypeScript
